## Как работает логика прохождения
Сайт проверяет параметр URL `?completed=aldar` и сохраняет завершение в `localStorage`.
После этого:
1. открывается модальное окно с итогами;
2. разблокируется кнопка «Изучить событие»;
3. разблокировка сохраняется между перезагрузками.

## Как подключить Unity WebGL позже
Сейчас в папке `game/` лежит демо-страница. Когда будет готов билд Unity WebGL, её можно заменить билдом.
Нужно только, чтобы после завершения эпизода страница возвращала пользователя на:
`../index.html?completed=aldar`

Если удобнее через JavaScript, можно выставить ключ вручную:
`localStorage.setItem('gg_completed_aldar', 'true')`
и затем перенаправить пользователя на главную страницу сайта.
